Eco Board Game

INSTRUÇÕES

    1° PASSO

  • Clique na bandeira verde para iniciar o jogo.

    • 2° PASSO

  • Clique no botão para continuar.

    • 3° PASSO

  • Tio Green irá explicar o que é o ECO BOARD GAME. Para continuar, clique na seta para prosseguir.

    • 4° PASSO

  • Aparecendo o menu inicial, você pode acessar as dicas disponíveis para os jogadores.

    • 5° PASSO

  • Após acessar as dicas e estiver tudo pronto, clique em PLAY para iniciar a partida. Assim, escolha o modo de jogo, CONTRA O COMPUTADOR ou CONTRA AMIGOS. Se escolher a opção "jogar contra amigos", escolha a quantidade de jogadores de no máximo 5 jogadores.

    • 6° PASSO

  • Digite o nome dos jogadores.

    • 7° PASSO

  • Após a contegem regressiva, clique para rodar os dados. Quem chegar ao final primeiro vencerá a partida.

Tabuleiro na versão simplificada para o jogo digital

Jogue o jogo incorporado abaixo

ECO BOARD GAME é um jogo de tabuleiro educativo, nas versões física e digital, que facilite o processo de ensino e aprendizagem sobre Educação Ambiental para os estudantes da educação básica, com o intuito de conscientizar a preservação do ecossistema adotando práticas sustentáveis e incentivando a geração atual a preservarem o meio ambiente para não comprometer os recursos naturais das gerações futuras.
